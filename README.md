# 📚 网课视频管理器

一个轻量级的本地网课视频管理工具，基于 Flask + Bootstrap 5 构建，帮助你管理桌面上的视频课程资源。

## ✨ 功能特性

- **视频扫描导入** — 自动扫描「网课视频库」文件夹中的视频文件，支持子文件夹结构
- **分类管理** — 自定义分类（如：编程、数学、英语），支持 Emoji 图标
- **标签系统** — 为视频添加任意标签，方便快速筛选
- **播放状态追踪** — 未看 / 在看 / 已看 三种状态，一目了然
- **网页播放器** — 直接在浏览器中播放视频，无需切换应用
- **系统播放器调用** — 一键用系统默认播放器打开视频
- **多种导入方式** — 文件上传、粘贴路径、扫描文件夹
- **搜索筛选** — 按分类、状态、标签、标题搜索
- **备注功能** — 为每个视频添加学习笔记

## 🛠 技术栈

| 组件 | 技术 |
|------|------|
| 后端 | Python 3 + Flask |
| 数据库 | SQLite |
| 前端 | Bootstrap 5 + Vanilla JS |
| 视频元数据 | ffprobe (FFmpeg) |
| 桌面启动 | pythonw.exe (无控制台窗口) |

## 📁 项目结构

```
网课管理器/
├── app.py              # Flask 应用入口
├── launcher.pyw        # 双击启动（无控制台窗口）
├── models.py           # 数据库模型与 CRUD
├── routes.py           # API 路由
├── scanner.py          # 视频扫描与元数据提取
├── templates/          # Jinja2 模板
│   ├── base.html       # 基础布局
│   ├── index.html      # 视频列表主页
│   ├── video_detail.html # 视频详情页
│   └── categories.html # 分类管理页
├── static/             # 静态资源 (Bootstrap, 图标, CSS)
└── data/               # SQLite 数据库文件
```

## 🚀 快速开始

### 环境要求

- Python 3.8+
- FFmpeg（用于获取视频时长信息，可选）

### 安装依赖

```bash
pip install flask
```

### 启动应用

**方式一：双击 `launcher.pyw`**（推荐，无控制台窗口）

**方式二：命令行启动**

```bash
cd "桌面\网课管理器"
python app.py
```

启动后浏览器会自动打开 `http://127.0.0.1:5000`。

### 安装 FFmpeg（可选）

FFmpeg 用于获取视频时长和编码信息，缺失时不影响基本使用。

- 下载地址：https://ffmpeg.org/download.html
- 或使用包管理器：`winget install ffmpeg` / `choco install ffmpeg`

## 📖 使用指南

### 导入视频

1. **推荐方式**：将视频文件复制到桌面的 `网课视频库` 文件夹，点击「扫描」按钮
2. **上传方式**：点击「导入」→「选择文件上传」
3. **路径方式**：点击「导入」→「粘贴文件路径」

### 管理视频

- 点击视频标题进入详情页
- 修改标题、分类、播放状态
- 添加标签和备注
- 支持网页播放和系统播放器打开

### 分类管理

点击侧边栏「管理分类」进入分类管理页面，支持增删改查。

## 📦 支持视频格式

`.mp4` `.mkv` `.avi` `.mov` `.wmv` `.flv` `.webm` `.m4v` `.mpg` `.mpeg` `.rmvb` `.ts` `.3gp`
